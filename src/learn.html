Quick Tab Switcher Guide

Welcome to **Quick Tab Switcher**! This extension helps you switch between tabs more efficiently by allowing you to index and access tabs with a few quick keyboard shortcuts.

How to Use

  1. Indexing Tabs: While on a tab you want to index, press Key + Tab. This will assign an index to the current tab.
  2. Switching to Indexed Tabs: To quickly access the tab you indexed, press Key + Shift. You’ll instantly jump to your indexed tab without scrolling through open tabs.

Features

  • **Easy tab indexing** with `Key + Tab` for fast access.
  • **Quick access** to indexed tabs with `Key + Shift`.
  • **Increased productivity** by reducing the time spent finding tabs.

Installation Instructions

  1. Download or clone the Quick Tab Switcher repository to your computer.
  2. Open Chrome and navigate to chrome://extensions/.
  3. Enable Developer Mode in the top right corner.
  4. Click on Load unpacked and select the extension folder.

Troubleshooting

If you encounter any issues, try the following:

  • Ensure you’re pressing the correct key combinations.
  • Reload the extension from the Extensions page (chrome://extensions/).
  • If the issue persists, please reach out to support at your-email@example.com.
